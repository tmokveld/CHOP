# CHOP: Phased path extraction from population graphs

## How to use

### Prerequisites

CHOP requires:

    - Python 2.7
    - networkx 1.9
    - bitarray

### Installing

    git clone https://github.com/tomokveld/CHOP
    cd CHOP
    python setup.py install

### Example usage

    chop -H -k 4 -p 1 -g examples/Figure_1.gfa.gz -o out.fa
